Ще більше можливостей для графіки

 This "Cloud of Points" graph is the first type of graph that can be created directly on Open Food Facts. Of course, you can create many more by downloading data and analyzing it with software like Open Office, Google Refine or Excel. But it seems very interesting to me to easily create graphics directly on Open Food Facts: 2 steps in less (the download of data and their import into an external software) make the creation of graphics much more accessible to the general public.

Developing graphing possibilities on Open Food Facts has another advantage: we can imagine features specifically adapted to food products. For example, I would like to add a graph to visualize the dyes used in products as a "pie chart", with the color of each slice calculated for each combination of dyes. For cheeses whose beautiful crust is colored with carmines (red) and annatto (yellow), we would have a nice orange slice.

Note: Before publishing a graph generated with the data, it is a good idea to look at the corresponding products at the most extreme points. An error in data entry is always possible, so if a value seems very high or very low, it's a good idea to double-check it.
