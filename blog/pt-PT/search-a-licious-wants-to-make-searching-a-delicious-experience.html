O Open Food Facts é, tanto quanto sabemos, a mais importante base de dados aberta sobre alimentos. Com cerca de 3 milhões de produtos, pode não ser fácil encontrar informações relevantes!

O projeto começou com capacidades de pesquisa bastante boas, oferecendo muitas agregações para explorar a base de dados, gráficos e capacidades de pesquisa avançadas (alguma vez o explorou?). Mas é bastante austero e carece de interatividade. Além disso, tem algumas limitações no que respeita à pesquisa de texto integral e às possibilidades de combinação.

Mas a situação vai melhorar! Graças ao programa NGI, Next Generation Internet, financiado pela Comissão Europeia, estamos a construir uma nova integração de pesquisa. Isto não beneficiará apenas o projeto Open Food Facts, uma vez que este pretende ser uma solução de pesquisa chave-na-mão fácil de integrar noutros projetos que lidam com grandes coleções de itens.

Esta pesquisa fornecerá:

  • bons recursos de texto completo,
  • lidar com precisão com vários idiomas,
  • oferecer refinamento por meio de pesquisas facetadas e sugestões,
  • aprimorar os recursos gráficos
  • e fazer tudo rapidamente!
  • Ele terá uma API poderosa e fácil de usar.
  • Isso também abrirá caminho para mais recursos, como compreensão inteligente de consultas, preenchimento automático, etc.

uma maquete de uma possível nova pesquisa

Ter uma pesquisa no mesmo nível dos melhores motores de busca permite que os dados abertos alcancem um público maior e obtenham mais impacto.

Ajuda as pessoas a encontrar produtos que correspondam à sua dieta ou necessidades específicas, permite encontrar melhores alternativas. Permite que investigadores, jornalistas ou ativistas investiguem rapidamente e descubram factos ocultos.

Também vemos este projeto como uma oportunidade para outros programadores configurarem rapidamente novas plataformas que aproveitem os conjuntos de dados disponíveis com características semelhantes.

Graças a Simon, já temos uma base para o projeto que vamos transformar completamente para cumprir os novos objetivos.
