Open Food Facts es, hasta donde sabemos, la base de datos abierta sobre alimentos más importante. ¡Con cerca de 3 millones de productos, puede que no sea fácil encontrar información relevante!

The project began with quite good search capabilities, offering a lot of aggregations to explore the database, graphics and advanced search capabilities (did you ever explore it?). Pero es bastante austero y carece de interactividad. También tiene una serie de limitaciones en la búsqueda de texto completo y posibilidades de combinación.

¡Pero la situación va a mejorar! Gracias al programa NGI, Next Generation Internet, financiado por la Comisión Europea, estamos construyendo una nueva integración de búsqueda. This won’t benefit the Open Food Facts project alone as it aims to be a turnkey search solution easy to integrate to other projects dealing with large collections of items.

Esta búsqueda proporcionará:

  • buenas capacidades de texto completo,
  • manejar con precisión múltiples idiomas,
  • offer refinement through faceted search and suggestions,
  • mejorar las capacidades gráficas
  • ¡y hacerlo rápidamente!
  • Vendrá con una API potente y fácil de usar.
  • It will also pave the way for more features like smart understanding of queries, auto-completion, etc.

una maqueta de una posible nueva búsqueda

Poseer un buscador a la altura de los mejores permite que los datos abiertos lleguen a un público más amplio y obtengan mayor impacto.

It helps people find products that correspond to their diet or specific needs, it enables finding better alternatives. Permite a investigadores, periodistas o activistas investigar rápidamente y descubrir hechos ocultos.

También vemos este proyecto como una oportunidad para que otros desarrolladores establezcan rápidamente nuevas plataformas que aprovechen los conjuntos de datos disponibles dadas sus características similares.

Gracias a Simon, ya tenemos una base para el proyecto que vamos a transformar completamente para alcanzar los nuevos objetivos. If you are a developper, a UI or UX designer, you are welcome to participate in this adventure. Even as a user, your use cases are welcome, don’t hesitate to communicate them.
